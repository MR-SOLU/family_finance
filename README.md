# family_expense  
🚀 💎 Family Finance Manager
Smart AI + Voice-Based Expense & Budget Tracking System
🌟 Overview

Family Finance Manager is a modern web application that helps users manage their daily finances with ease using:

💰 Expense tracking
📊 Budget management
🤖 AI-based financial insights
🎤 Voice input support

All wrapped inside a premium dark-themed dashboard UI.

🎯 Purpose of the Project

Managing money manually is:

Time-consuming
Error-prone
Lacking insights

This project solves that by providing:
👉 A smart financial assistant that tracks, analyzes, and guides spending behavior.

💡 Features
💰 Expense Management
Add expenses (amount, category, date)
View all transactions
Track daily & monthly spending
📊 Budget Tracker
Set monthly/category budgets
Monitor usage in real-time
Visual progress tracking
📈 Dashboard
Total balance overview
Expense summaries
Category-wise insights
Clean analytics view
🤖 AI Finance Assistant
Answer queries like:
“How can I save money?”
“Where am I overspending?”
Provides smart suggestions
🎤 Voice Input (Smart Feature)
Add financial data using voice
Example:
“I spent 500 on food”
Automatically extracts:
Amount
Category
🎨 Premium UI
Dark mode interface
Modern dashboard layout
Smooth interactions

⚙️ Installation
1. Clone Repo
git clone https://github.com/MR-SOLU/family_finance.git
cd family_finance
2. Install Dependencies
npm install
cd client
npm install
3. Run Project
npm run dev
